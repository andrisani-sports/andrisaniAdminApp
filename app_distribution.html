Shoulder Saver App

Before installing the application on a mobile device, do the following...

  1. Be connected to the Internet on the target mobile device
  2. Have a file explorer app installed on the target mobile device. Here are a couple of options
    • (If you are installing on an Amazon Kindle Fire) File Explorer
    • (For other Android devices) File Explorer+
  1. Open this page on the browser in the target mobile device
  2. Click here to get the app and download onto the device
  3. Open your file explorer app
  4. Go to the "Downloads" folder, the file "android-debug.apk" should be in there, and click on that file
  5. Click on "Install", then click on "Open"
